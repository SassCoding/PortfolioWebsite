RRC Automotive Group - C# Vehicle Sales Quote Calculator

My first major project of the Business Information Technology program. Our learning goal for the second term of programming after learning Java was to jump into the world of C# and Windows Forms. This application required creating data-bound properties and invoice forms that used a database populated with vehicle information that could be accessed and changed via Windows Form. I am proud of this project because it was my first glimpse into the project space. I got to see from start to finish how applications are created, rigorously tested, and debugged before completion.

iMovieRatings - Content Management System - *In Progress*

iMovieRatings is a project of mine that I am continuing to work on. It is a CMS website created using PHP/HTML/CSS/BootStrap and MyPHPAdmin. Using a fully working login system, users can sign up, and browse for movies they wish to read reviews on, or post ratings/reviews of their own. Reviews can be edited/deleted by the user who posted them and the changes are all reflected by the database. This project has been one of my favourites which is the reason I have continued to work on it on my own time.
